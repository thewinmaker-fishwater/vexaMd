# Vexa MD 플러그인 사용자 가이드

이 가이드는 Vexa MD 플러그인을 설치하고 관리하는 방법을 설명합니다.

---

## 플러그인이란?

플러그인은 Vexa MD의 기능을 확장하는 추가 모듈입니다. 예를 들어 Mermaid 플러그인은 마크다운 코드 블록에서 다이어그램을 렌더링합니다.

---

## 플러그인 관리 열기

1. 툴바에서 **플러그인** 버튼 클릭
2. 플러그인 관리 모달이 열립니다

---

## 내장 플러그인

Vexa MD에는 기본 제공되는 내장 플러그인이 있습니다. "내장" 배지로 표시됩니다.

### Mermaid Diagrams
- 마크다운에서 Mermaid 다이어그램을 렌더링합니다
- 설정: 테마, 보안 수준

---

## 플러그인 활성화/비활성화

1. 플러그인 관리 모달 열기
2. 플러그인 카드의 **토글 스위치**를 클릭
3. 활성화 시 플러그인이 즉시 동작합니다

---

## 플러그인 설정

1. 활성화된 플러그인 카드에서 **설정** 버튼 클릭
2. 설정 모달에서 옵션 변경
3. **적용** 버튼 클릭으로 저장
4. **초기화** 버튼으로 기본값 복원

설정 타입:
- **토글**: 켜기/끄기
- **선택**: 드롭다운 목록에서 선택
- **텍스트**: 직접 입력
- **숫자**: 숫자 입력
- **색상**: 컬러 피커
- **범위**: 슬라이더

---

## 플러그인 설치

외부 플러그인을 설치하려면:

1. 플러그인 관리 모달에서 **플러그인 설치** 버튼 클릭
2. 플러그인이 들어있는 폴더 선택 (plugin.json 파일이 있어야 함)
3. 자동으로 앱 데이터 디렉터리에 복사됨
4. 플러그인 목록에 "사용자 설치" 배지로 표시됨

### 요구사항
- 플러그인 폴더에 `plugin.json` 파일이 있어야 합니다
- `plugin.json`에 `id`, `name`, `version`, `main` 필드가 있어야 합니다

---

## 플러그인 삭제

사용자 설치 플러그인만 삭제할 수 있습니다 (내장 플러그인은 삭제 불가).

1. 플러그인 카드에서 **삭제** 버튼 클릭
2. 확인 다이얼로그에서 **확인** 클릭
3. 플러그인 파일이 삭제되고 목록에서 제거됨

---

## 문제 해결

### 플러그인 로드 실패

에러가 발생한 플러그인은 붉은 테두리로 표시됩니다.

1. **에러 상세**를 클릭하여 에러 내용 확인
2. **재시도** 버튼으로 다시 로드 시도
3. 문제가 지속되면 플러그인 개발자에게 문의

### 플러그인이 목록에 안 보임

- 플러그인 폴더에 `plugin.json`이 있는지 확인
- `plugin.json`의 형식이 올바른지 확인
- 콘솔(F12)에서 에러 메시지 확인

### 설정이 저장되지 않음

- 플러그인이 활성화되어 있는지 확인
- 브라우저 localStorage가 가득 차지 않았는지 확인

---

## 플러그인 저장 위치

- **내장 플러그인**: 앱 설치 디렉터리 내 `src/plugins/`
- **사용자 설치 플러그인**: `%APPDATA%/com.vexa-md/plugins/`
